Добавление карточек и наборов
=============================

1. Глаголы во всех временах и местоимениях
-----------------------------------------
Файлы наборов лежат в каталоге `sets/`.
Чтобы добавить новую подборку:
- создайте файл `sets/имя_набора.js` по примеру существующих;
- опишите объект с полями `title`, `description` и массивом `verbs`;
- экспортируйте этот объект `export default { ... }`;
- добавьте новый импорт в `sets/index.js` и включите его в массив экспортов;
- выполните `node build.mjs`, чтобы обновить `bundle.js`.

2. Карточки
-----------
Отдельные наборы карточек находятся в папке `cards/` в файлах вида
`cards/название.js` и содержат массив `CARDS`.
Чтобы добавить новую страницу с карточками:
- создайте файл с массивом `CARDS`;
- сделайте HTML‑страницу по примеру существующих (например `adverbs.html`),
  подключив в ней `flashcards.js` и ваш файл из `cards/`;
- добавьте ссылку на новую страницу в меню в функции `renderMenu()` файла `main.js`.

3. Словарный тренажёр
---------------------
Наборы слов для тренажёра также лежат в `cards/` и подключаются в `words.js`.
Чтобы добавить новый набор:
- создайте файл `cards/название.js` с `const CARDS = [...]`;
- добавьте информацию о наборе в объект `decks` в `words.js` (ключ, `title` и `src`);
- добавьте ссылку на `words.html?set=<ключ>` в меню `renderMenu()` в `main.js`.
